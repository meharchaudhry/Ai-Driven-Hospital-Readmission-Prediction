# Ai-Driven-Hospital-Readmission-Prediction


This project builds machine learning models to predict hospital readmission risk using patient
demographics, lab results, and hospital usage data.

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Tools & Libraries
Python, Pandas, NumPy, scikit-learn, Tableau

## Results
- Achieved 98.2% accuracy in predicting readmission risk
- Built an interactive dashboard to identify high-risk patients

## Use Case
Supports early intervention and data-driven decision making in healthcare.
